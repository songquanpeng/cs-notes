# Gnome 设置

首先务必安装 Gnome Tweak Tools

# 配置

  1. Setting -> Display
    • 2K 27 寸屏幕缩放 125%（启用 Fractional Scaling）。
    • 最上方的 Night Light，调节 Color Temperature 为约 1/5 处。
  2. Setting -> Appearance：Icon Size 取 30

主题选择 Mojava-light
