# 环境变量

  1. 本行命令有效：ENV_VAR=value command。
  2. 本次会话有效：export ENV_VAR=value。
  3. 永久有效：将 export ENV_VAR=value 加入 ~/.bashrc 中。

PATH 环境变量：export PATH=$PATH:/to/the/path
